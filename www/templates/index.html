The DTMF Robot generates DTMF command sequences needed to program an S-COM 5K/6K/7K repeater controller, or for other situations where you need to maintain long/complex DTMF commands. Simply upload your existing S-COM Programmer command script, press GENERATE, download the audio file (in .WAV format), and play the file into your repeater controller.

Information about the original S-COM Programmer (written by Dave Maciorowski WA1JHK) and the methods of playing DTMF streams into S-COM controllers can be found here on the Repeater Builder website.

This tool is provided for amateur radio use and is not an S-COM product. Questions and/or suggestions should be sent to the author KC1FSZ (good in QRZ). Please don't bother S-COM with support requests!
